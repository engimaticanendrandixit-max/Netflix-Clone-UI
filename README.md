# Netflix Clone 🎬

A Netflix-inspired frontend clone built using HTML, CSS, and JavaScript.  
The project recreates the look and feel of Netflix's landing page with a modern UI design.

## 🚀 Features Completed

- Netflix-style navigation bar
- Hero section with background banner
- Call-to-action buttons
- Login section UI
- Curved separator design
- FAQ section
- Footer section
- Responsive layout

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
